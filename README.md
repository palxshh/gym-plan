# Gym Plan 🏋️

A mobile-first, beginner-friendly workout web app — a **4-day Upper/Lower body-recomposition** plan with animated exercise demos, form cues, a warm-up routine, nutrition basics, an on-device progress tracker and an in-gym rest timer.

**Live:** https://palxshh.github.io/gym-plan/

## Features
- **4-day split** — Push · Lower A · Pull · Lower B, in a colour-coded Workout hub with a day-switcher
- **Animated demo** + tempo, breathing and form tips (and a common-mistake warning) for every exercise
- **Warm-up** routine, **protein calculator**, **progress tracker** (saved on your device) and a **rest timer**
- Single self-contained `index.html` — no build step, no dependencies (Google Fonts CDN only)
- Works offline once loaded; "Add to Home Screen" for an app-like experience
